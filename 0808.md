# 파일설치!

> <h4>APACHE, TOMCAT</h4>
>
> <H5>APACHE</H5>
>
> Apache 재단에서 관리하는 HTTP 웹서버로써 무료 오픈소스이며, 구축이 쉽다는 이유로 세계에서 많이 쓰이는 웹서버이다.
> * CentOS 에서 아파치를 설치하는 방식
>   * 소스파일을 다운로드하고 컴파일하여 설치하는 컴파일 설치방식
>   * RPM 패키지 파일을 YUM 방식(의존성해결됨)으로 설치하는 방식

> <h5>OpenSSL</h5>
>
>  웹 브라우저와 서버간의 통신을 암호화하는 오픈소스 라이브러리

> <h5>TOMCAT</h5>
>
> - Apache 재단에서 관리하는 오픈소스 프로젝트이며 웹컨테이너만 있는 웹 어플리케이션 서버(WAS)이다.
> - EJB컨테이너 및 JAVA EE 스펙등을 포함한 다른 WAS에 비해 기능은 적으나 오픈소스이며 사용법이 쉽기때문에 널리 쓰인다.
> - Tomcat은 웹서버와 연동하여 실행할 수 있는 자바 환경을 제공하여 자바 서버페이지(JSP)와 자바 서블릿이 실행할 수 있는 환경을 제공하고 있다.
> - Tomcat은 관리툴을 통해 설정을 변경할 수 있지만 XML 파일을 편집하여 설정할 수도 있으며 HTTP 서버도 자체내장한다.
