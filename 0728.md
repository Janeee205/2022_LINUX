> <h4> 파일, 폴더 생성/보기/삭제</h4>

> * **cat > a1**  : 내가 입력한 글을 a1이라는 파일로 생성, 저장
> * **cat a1** : 파일 a1 보기
> * **cat a1 > a2** : a2를 생성하여 a1의 내용을 복사(덮어씀), 저장
> * **cat a1 >> a2** : 기존의 a2의 내용에 a1의 내용을 추가하여 저장
> * **tac a2** : a2의 내용을 거꾸로 보기
> * **rm a2** : a2삭제
> * __rm *.txt__ : txt확장자의 파일 전부 삭제
> * 
> * **mkdir cmd** : cmd라는 이름으로 폴더 생성
> * **rmdir cmd** : cmd 폴더 삭제 (빈 폴더만 가능)
> * **rm -rf cmd** : 빈폴더 아니어도 삭제됨
> * **cd cmd** : cmd 폴더로 이동
> * **cm ../** : 상위 폴더로 이동
>  #
> * _nl 은 cat 과 유사하나 앞에 줄번호가 붙어있다._
> * **nl a1 = cat -n a1** (-n은 줄번호)
> * **nl -w10 al** : 앞에 10칸만큼 공백 넣어 출력
> * **nl -i2 a1** : 줄번호 2씩 증가
> * **nl -v2 -i3 a5 : 첫 줄번호 2에서 시작, 3씩 증가**


> <h4> cal 명령어</h4>

> * cal : 달력
> * **cal > a1** : 달력을 a1이라는 파일로 생성하여 저장
> * **cal > a3.txt** : txt파일로 a3 저장
